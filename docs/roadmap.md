Roadmap
=======

V0.1
----

- Python-first developer API.
- Standard JSON response and error format.
- Threaded listener with access logs.
- CLI: `run`, `serve`, `dev`, `docs`, `routes`, `bench`, `new`.
- ML guard with intent, feature, anomaly, risk, bot and action scoring.
- Streaming file response helpers for static assets and downloads.
- In-process job queue with status and cancel endpoints.
- Ecommerce response presets.
- C/C++ native core starter library with stable C ABI and security/risk hotpath helpers.

Next
----

- Native route matching bridge from Python.
- Request/response serialization hot path in C++.
- Durable production worker model.
- Reload mode for development.
- Structured JSON logs.
- More complete OpenAPI schema generation.
- Multipart/streaming upload parser.
- ORM, migrations and auth middleware.
